# Sample Sanctions Data

These are **synthetic, illustrative sample lists** for development and demo only.
Every entity is prefixed `SAMPLE -` and is fabricated. They are NOT real OFAC/EU/UN
sanctions lists and must not be used for any real screening.

Files:
- ofac_sdn_sample.csv, eu_consolidated_sample.csv, un_consolidated_sample.csv — entity lists
- country_risk_sample.csv — country risk tiers + embargo flags
- dual_use_keywords_sample.csv — dual-use goods keyword triggers
